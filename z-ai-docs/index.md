# Z.AI Documentation

Z.AI offers a platform for accessing multiple AI models including GLM-4.7, GLM-4.6V, GLM-Image, and CogVideoX-3 through various development methods.

## Available Models

| Model | Description | Type |
|-------|-------------|------|
| [GLM-4.7](guides/models/glm-4.7.md) | Latest flagship model with SOTA performance and superior Agentic Coding | LLM |
| [GLM-4.6V](guides/models/glm-4.6v.md) | Multimodal model with 128K context for vision tasks | VLM |
| [GLM-Image](guides/models/glm-image.md) | Text-to-image generation | Image |
| [CogVideoX-3](guides/models/cogvideox-3.md) | Video generation with frame interpolation | Video |

## API Endpoints

**General API Endpoint:**
```
https://api.z.ai/api/paas/v4
```

**Dedicated Coding Endpoint:**
```
https://api.z.ai/api/coding/paas/v4
```

## SDK Compatibility

Z.AI supports multiple calling methods:
- HTTP API (RESTful)
- Official Z.AI Python SDK
- Official Z.AI Java SDK
- OpenAI-compatible Python SDK
- OpenAI-compatible Node.js SDK
- OpenAI-compatible Java SDK
